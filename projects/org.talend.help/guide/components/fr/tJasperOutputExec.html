tJasperOutputExec

Propriétés du tJasperOutputExec

Ce composant est étroitement lié à l'outil de reporting de Jaspersoft, iReport. Il lit et traite des données d'un fichier afin de créer un rapport à l'aide d'un modèle de rapport .jrxml défini via iReport. Ce composant permet un gain de performance, car il est la combinaison d'un composant d'entrée et d'un composant tJasperOutput. L'utilisation de ces deux composants séparés permet de transformer les données avant de les utiliser pour générer un rapport, avec des sources riches et variées en entrée.

Famille de composant

Business Intelligence/Jasper

 

Fonction

Le composant tJasperOutputExec lit et traite des données à partir d'un fichier source pour créer un rapport à l'aide d'un modèle de rapport .jrxml défini via iReport.

Objectif

Ce composant vous permet d'utiliser l'outil iReport de Jaspersoft afin de créer un rapport dans des formats riches. Il permet un gain de performance, puisqu'il est la combinaison d'un composant d'entrée et du composant de sortie tJasperOutput.

Basic settings

Jrxml file

Fichier du modèle de rapport créé dans iReport

 

Source file

Nom du fichier source.

 

Record delimiter

Saisissez votre délimiteur d'enregistrements.

 

Destination path

Chemin d'accès au fichier final du rapport.

 

Use Default Output Name

Cochez cette case pour utiliser le nom par défaut du rapport généré, c'est-à-dire le nom du fichier source.

 

Output Name

Nom du rapport final.

Ce champ n'apparaît pas si la case Use Default Output Name est cochée.

 

Report type

Type du fichier du rapport final.

 

iReport

Editez la commande pour fournir le chemin d'accès au fichier d'exécution de iReport, par exemple en remplaçant __IREPORT_PATH__\ par E:\Program Files\Jaspersoft\iReport-4.1.1\bin\, ou en saisissant le chemin d'accès complet au fichier d'exécution, par exemple "E:\Program Files\Jaspersoft\iReport-4.1.1\bin\iReport.exe".

 

Launch

Cliquez sur ce bouton pour lancer iReport.

Advanced settings

tStatCatcher Statistics

Cochez cette case pour collecter les données de log au niveau du composant.
