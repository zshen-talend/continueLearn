tJDBCTableList

Propriétés du tJDBCTableList

Famille de composant

Databases/JDBC

 

Fonction

Le composant tJDBCTableList effectue une opération d’itération sur toutes les tables d’une base de données, grâce à une connexion JDBC définie.

Objectif

Le composant tJDBCTableList permet de lister les noms des tables JDBC grâce aux commandes SELECT et WHERE.

Basic settings

Component list

Sélectionnez le composant de connexion tJDBCConnection dans la liste si vous prévoyez d’ajouter plus d’une connexion à votre Job en cours.

 

Where clause for table name selection

Saisissez la commande WHERE permettant d’identifier les tables sur lesquelles effectuer l’opération d’itération.

Utilisation

Il faut utiliser ce composant en association avec les autres composants JDBC, notamment avec le tJDBCConnection.

Limitation

n/a

Scénario associé

Pour un scénario associé, consultez la section intitulée « Scénario : Itérer une table de base de données et lister le nom des colonnes de la table ».
