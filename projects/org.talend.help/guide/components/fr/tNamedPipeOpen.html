tNamedPipeOpen

Propriétés du tNamedPipeOpen

Famille de composant

File/Input

Fonction

Le composant tNamedPipeOpen ouvre un tube nommé pour écrire des données à l’intérieur.

Objectif

Ce composant est utilisé dans un processus de communication interne, il ouvre un tube nommé pour écrire des données à l’intérieur.

Basic settings

Name

Saisissez dans le champ le nom du tube nommé.

 

Delete if already exist

Cochez cette case si vous ne souhaitez pas dupliquer un tube nommé. Le tube nommé existant sera remplacé.

Advanced settings

tStatCatcher Statistics

Cochez cette case pour collecter les données de log au niveau du Job, ainsi qu’au niveau de chaque composant.

Utilisation

Ce composant est généralement utilisé comme composant de début dans un Job effectuant un processus de communication interne.
