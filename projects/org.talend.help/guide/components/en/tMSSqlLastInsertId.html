tMSSqlLastInsertId

tMSSqlLastInsertId properties

Component Family

Databases/MS SQL server

 

Function

tMSSqlLastInsertId displays the last IDs added to a table from a MSSql specified connection.

Purpose

tMSSqlLastInsertId enables you to retrieve the last primary keys added by a user to a MSSql table.

A schema is a row description, i.e., it defines the number of fields to be processed and passed on to the next component. The schema is either built-in or stored remotely in the Repository.
