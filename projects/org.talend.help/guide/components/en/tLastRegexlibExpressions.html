This component will be available in the Palette of the studio on the condition that you have subscribed to the relevant edition of Talend Enterprise Data Quality.

tLastRegexlibExpressions

Component family

Data Quality

 

Function

tRegexlibExpressionsconnects to a web service at http://regexlib.com to get the N most recent expressions from the list of regular expressions.

Purpose

tFindRegexlibExpressionsreturns a dataset holding information about the N most recent regular expressions added to the library and that match the query at http://regexlib.com.

Basic settings

Schema and Edit schema

These fields are read-only. The output schema of this component contains the following fields: Title, Expression, Description, Matches, Non-matches, Author, Rating.

 

Max Rows

Enter the max number of the expressions you want to get.

Advanced settings

tStatCatcher Statistics

Select this check box to collect log data at the component level.

Usage

This component is a start component. It requires an output flow, usually a csv file to be able later to import the collected expressions into Talend Enterprise Data Quality studio.

For more information about importing patterns, see Talend Enterprise Data Quality User Guide.
