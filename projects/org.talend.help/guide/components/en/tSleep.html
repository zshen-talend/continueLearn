tSleep

tSleep Properties

Component family

Orchestration

 

Function

tSleep implements a time off in a Job execution.

Purpose

Allows you to identify possible bottlenecks using a time break in the Job for testing or tracking purpose. In production, it can be used for any needed pause in the Job to feed input flow for example.

Basic settings

Pause (in second)

Time in second the Job execution is stopped for.

Usage

tSleep component is generally used as a middle component to make a break/pause in the Job, before resuming the Job.
