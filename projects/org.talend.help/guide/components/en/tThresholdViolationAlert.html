tThresholdViolationAlert

This component will be available in the Palette of the studio on the condition that you have subscribed to the relevant edition of Talend Enterprise Data Quality.

This component is only usable with a Job which has generated reports.

tThresholdViolationAlert properties

Component family

Data Quality

 

Function

tThresholdViolationAlert can alert to any threshold violations regarding the thresholds set on indicators in different analyses created in Talend Enterprise Data Quality studio.

Purpose

tThresholdViolationAlert can communicate with Talend Enterprise Data Quality in order to check on thresholds put on the indicators used in analyses created in Talend Enterprise Data Quality. You can then use it with other components, for example tSendMail, to alert to the violation of thresholds.

For information on how to generate a Job to alert to threshold violation directly from Talend Enterprise Data Quality studio, see Talend Enterprise Data Quality User Guide.
