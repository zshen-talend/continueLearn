tVerticaRollback

tVerticaRollback Properties

This component is closely related to tVerticaCommit and tVerticaConnection. It usually does not make much sense to use these components independently in a transaction.

Component family

Databases/Vertica

 

Function

tVerticaRollback cancels the transaction commit in the connected DB.

Purpose

tVerticaRollback avoids to commit part of a transaction involuntarily.

Basic settings

Component list

Select the VerticaConnection component in the list if more than one connection are planned for the current job.

 

Close Connection

Clear this check box to continue to use the selected connection once the component has performed its task.

Advanced settings

tStatCatcher Statistics

Select this check box to collect log data at the component level.

Usage

This component is to be used along with Mysql components, especially with tVerticaConnection and tVerticaCommit components.

Limitation

n/a
